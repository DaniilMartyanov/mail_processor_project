# mail_processor_project

## Автоматизированная система обработки корпоративной почты, групповой проект в рамках хакатона по курсу «Технологии программирования».

## Цель проекта:  
Автоматизация процесса первичной обработки и сортировки входящих обращений в службу IT-поддержки.

## Структура проекта:  
* `inbox/` - папка, куда складываются исходные письма (нужно вставить Ваши письма).
* `mail_processor/` - логика приложения (парсер писем, движок правил фильтрации и перемещение файлов).
  * `models.py` - класс `Email`, отвечающий за чтение, кодировки и извлечение заголовков.
  * `rules.py` - движок фильтрации `RuleEngine`.
  * `engine.py` - координатор `EmailProcessor`.
  * `exceptions.py` - пользовательские исключения для обработки ошибок.
* `tests/` - тесты на pytest, проверяющие корректность парсинга и крайние случаи.
* `main.py` - точка входа для запуска обработки.
* `rules.json` - файл с ключевыми словами для сортировки.
* `run.sh` - bash-скрипт для быстрой очистки старых результатов и запуска нового цикла обработки.

## Требования:  
Вся основная логика реализована на стандартной библиотеке Python, зависимости требуются только для запуска тестов (pytest).

## Как запустить  
1. Клонируйте репозиторий и перейдите в корень проекта
2. Запустите приложение через bash-скрипт `./run.sh` в корне проекта
